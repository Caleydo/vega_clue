=====================

[Vega examples](https://vega.github.io/vega/examples/) with [CLUE](https://github.com/phovea/phovea_clue)

Installation
------------

```
git clone https://github.com/Caleydo/vega_clue.git
cd vega_clue
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```

Launching
---------

```
npm start
```
